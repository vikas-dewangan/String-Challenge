open terminal and follow the below step to test
1 first check the application
 -> type gradle clean build in terminal if gradle is working properly or not.
2. type gradle test, it will test the Main file for given test cases.
3. After running tests succesfully you can check the report in build folder
4. open the build folder -> report folder -> index.html
 You can serve this index.html or open in any browser to check the test report and results
